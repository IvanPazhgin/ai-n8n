# AI-N8N Project

Проект для автоматизации работы ИИ-агента по недвижимости с использованием n8n (workflow automation tool).

## Описание проекта

Этот проект содержит конфигурацию для локального запуска n8n с PostgreSQL базой данных через Docker. Основная цель - создание автоматизированных рабочих процессов для обработки запросов от потенциальных клиентов недвижимости.

## Структура проекта

```
.
├── docker-compose.yml          # Конфигурация Docker Compose
├── .env                        # Переменные окружения для Docker
├── .env.example                # Шаблон переменных окружения
├── promts/                     # Промпты для ИИ-агента
│   └── p1.md                  # Промпт для агента по недвижимости
├── help.md                     # Файл помощи
├── query                       # Сервисные файлы
├── start                       # Сервисные файлы
└── .idea/                      # Настройки IntelliJ IDEA
```

## Цель проекта

Создание ИИ-агента, который:

- Работает как профессиональный брокер по недвижимости
- Общается с клиентами на платформах Avito и Cian
- Использует специальные стратегии для получения контактных данных клиентов
- Автоматизирует процесс лидогенерации

## Основные правила агента

1. **Не отправлять материалы через чат** - планировки, фото, документы
2. **Не предлагать аналогичные объекты** - работать только с текущим запросом
3. **Использовать "крючки"** - экспертная ориентация, экономия времени, выгода, полнота информации
4. **Обрабатывать возражения** - настаивать на личной встрече/звонке
5. **Главная цель** - получить номер телефона клиента

## Локальный запуск

### Предварительные требования

- Docker Desktop (версия 26.0.0 или выше)
- Git

### Запуск проекта

1. Клонируйте репозиторий:

```bash
git clone https://github.com/IvanPazhgin/ai-n8n.git
cd ai-n8n
```

2. Убедитесь, что Docker Desktop запущен

3. Запустите сервисы:

```bash
docker-compose up -d
```

4. Дождитесь загрузки образов и запуска контейнеров

5. Откройте браузер и перейдите на http://localhost:5678

6. Войдите с учетными данными:
   - **Логин:** admin
   - **Пароль:** adminpassword123

### Проверка статуса

Проверить состояние контейнеров:

```bash
docker-compose ps
```

Остановить сервисы:

```bash
docker-compose down
```

## Переменные окружения

| Переменная              | Описание                   | Значение по умолчанию            |
| ----------------------- | -------------------------- | -------------------------------- |
| POSTGRES_USER           | Пользователь PostgreSQL    | n8n                              |
| POSTGRES_PASSWORD       | Пароль PostgreSQL          | strongpassword123                |
| POSTGRES_DB             | Имя базы данных            | n8n                              |
| N8N_BASIC_AUTH_USER     | Логин для n8n              | admin                            |
| N8N_BASIC_AUTH_PASSWORD | Пароль для n8n             | adminpassword123                 |
| N8N_ENCRYPTION_KEY      | Ключ шифрования (32+ байт) | 12345678901234567890123456789012 |
| WEBHOOK_URL             | URL для вебхуков           | http://localhost:5678            |
| GENERIC_TIMEZONE        | Часовой пояс               | UTC                              |

## Архитектура

Проект использует:

- **n8n** - платформа для автоматизации рабочих процессов
- **PostgreSQL** - реляционная база данных для хранения данных n8n
- **Docker Compose** - оркестрация контейнеров
